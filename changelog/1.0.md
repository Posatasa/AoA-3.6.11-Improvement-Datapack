# Changelog

## v1.0 – Initial Balance Update

### Mob Balance Changes

- Reduced max health of most hostile mobs across tiers:
  - **T0 mobs**: -50%
  - **T1 mobs**: -68%
  - **T2 mobs**: -72%
  - **T3 mobs**: -76%
  - **T4 mobs**: -80%
- Certain particularly frustrating mobs have had their health reduced by **90%**.

These changes aim to smooth early-to-mid game progression and reduce excessive time-to-kill without trivializing combat mechanics.

---

### Bug Fixes

- Fixed an issue where vanilla **Guardians** were not dropping **Biogenic Armor** as intended.

---

### Recipe Changes

- Added a new crafting recipe for **Poison Rune** using **Spider Eye**, making the rune more accessible in early progression.

---

### World Generation Adjustments

- Structures in the **Mysterium** dimension have been moved to the surface instead of generating underground.
  
  This change improves exploration flow and reduces unnecessary digging while preserving structural integrity.
